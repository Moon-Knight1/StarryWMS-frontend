  风格参考：Victorian Conservatory（维多利亚温室/标本馆风格）

- 底色：深墨绿 #111c15，卡片 bg 略浅 #17231b
- 主色：铜锈色 #c8956c，强调色：鼠尾草绿 #8fb89a，金色 #d4a857，奶油色 #e8dfd0
- 装饰：四角 copper 色 20px L 形装饰线，极淡植物/花卉 SVG 底纹（透明度 < 0.03）
- 字体：标题用 Libre Baskerville 斜体 + 衬线，正文用 DM Sans，数据用 JetBrains Mono 等宽
- 卡片：1px 铜色半透明边框（rgba(200,149,108,0.08)），hover 时变深、边框变亮
- 卡片入场动画：fade-up translateY(10px)，stagger delay 0.04s * index
- 数据标签：全大写、宽字距 letter-spacing 0.12em+、11-12px
- 图表：ECharts Nightingale Rose 玫瑰图（铜绿 palette）+ Gauge 模拟仪表盘，与整体复古风格一致
- 动效尊重 prefers-reduced-motion
- 所有值采用 CSS 自定义属性（--copper, --sage 等）
